sendMsgSyncStore

  • sendMsgSyncStore<T>(module: T, data: any, opt?: DatabaseSubModuleNames<T>): void
  • sendMsgSyncStore<T>(module: T, data: any, opt?: DataSyncOpt): void

  • 广播数据同步

    引入版本

    v0.0.1

    Type parameters

    • T: "__store__" | "pomeloState" | "windowState" | "systemState" | "commonState" | "flowState"

    Parameters

    • module: T

      模块名（如：pomeloData）

    • data: any

      要同步的数据

    • Optional opt: DatabaseSubModuleNames<T>

      配置项

      import { broadcast } from '@peiyou/eevee-helper';
const database = reactive({
     pomeloState: {},
     windowState: {}
})
// 同步整个模块
broadcast.sendMsgSyncStore('windowState', { winSizeType: 'normal' })

// 同步模块下某个属性
broadcast.sendMsgSyncStore('windowState', 'normal', 'winSizeType')
broadcast.sendMsgSyncStore('windowState', 'normal', { key: 'winSizeType' })

// 同时修改多个key
broadcast.sendMsgSyncStore('pomeloState', 'start', ['mainStatus', 'groupStatus'])
broadcast.sendMsgSyncStore('pomeloState', 'start', { key: ['mainStatus', 'groupStatus'] })

// 不同步当前窗口
broadcast.sendMsgSyncStore('windowState', 'normal', {
     key: 'winSizeType',
     isSyncCur: false // 默认true
})

 * // 只同步当前窗口
broadcast.sendMsgSyncStore('windowState', 'normal', {
     key: 'winSizeType',
     onlyCurrent: true // 默认false
})

// 同步整个database
broadcast.sendMsgSyncStore('__store__', null, { isSyncAllStore: true })

    Returns void

  • Type parameters

    • T: "__store__" | "pomeloState" | "windowState" | "systemState" | "commonState" | "flowState"

    Parameters

    • module: T
    • data: any
    • Optional opt: DataSyncOpt

    Returns void

Const unsub

  • unsub(type: string, fn?: Fn<any, any>): void

  • 取消监听/订阅频道事件（类似Vue - off） 注意： 未传入第二个参数：则会销毁掉所有窗口的 type 事件所有回调 传入第二个参数：则会销毁掉当前窗口的 type 事件对应的 fn 回调 查看详情unsub

    ```typescript import { broadcast } from '@peiyou/eevee-helper'; // 取消事件所有回调 broadcast.unsub('getUserList')

    // 取消事件对应回调 const cb = () => { ... } broadcast.unsub('getUserList', cb) ```
